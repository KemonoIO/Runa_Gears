Runa Gears

Summary

This package is spinning gear armlets and anklets for the Runa model by Torinyan. You must own the Runa base model to use it.

Installation

  1. Unpack the dependencies, in order.
  2. Unpack Runa_Gears.unitypackage.
  3. Drag Assets/Kemono_Io/Runa_Gears.prefab into the Runa_Source object in the Hierarchy panel.
  4. Set the Runa_Gears' anchor override.
    1. Select the Runa_Gears mesh in the Hierarchy panel.
    2. Drag the hips bone into the Anchor Override in the Inspector panel.
  5. Constrain the Runa_Gears's armature to the Runa base model's armature.
    1. Fully expand the Runa_Gears object's armature in the Hierarchy panel.
    2. Select one of the bones, such as hips.
    3. Drag the identically-named bone from the Runa base model's armature to the Parent Constraint field in the Inspector panel.
    4. Click the Activate button.
    5. Repeat the last three steps for each bone in the armature.
  6. Merge the FX animator controllers.
    1. Open the Avatars 3.0 Manager.
    2. Select the Runa_Source object and expand the FX animator.
    3. Click the "Add animator to merge" button.
    4. Select the Runa_Gears_Fx animator and click the "Merge on current" button.
